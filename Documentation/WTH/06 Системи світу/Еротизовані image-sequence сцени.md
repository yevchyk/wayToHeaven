# Еротизовані image-sequence сцени

Еротизовані або інтимно-небезпечні сцени в цьому проєкті не повинні authorитися як звичайні діалогові портретні сцени. Це canonical reminder, що проєкт працює як доросла темна еротична новела, а не як дитяча казка з "пікантними" винятками.

## Контракт

Такі сцени:
- не є `character dialogue blocking`
- не покладаються на стандартну схему `героїня ліворуч / співрозмовник праворуч`
- мають іти через `background / CG / image-state` зміни, narration і scene-level transitions
- можуть свідомо очищати `stage`, якщо портретна постановка руйнує сенс спостереження або неправильного зближення

## Поточні сцени

Поточний перелік нижче є canonical registry для наявного runtime-контенту прологу.

- `chapter-1/scene/intro :: s7_betrayal -> s7_betrayal_body`
  note: [[Зрада]]
- `chapter-1/scene/intro/two-of-three :: a_004_evening -> a_007_exit`
  note: [[Вечір у кімнаті Мірелли]]

## Не плутати

- suggestive beats на кшталт ранкової шнурівки чи побутової тілесної напруги ще не обов'язково є adult image-sequence сценою
- adult image-sequence починається там, де сцена вже читається через образ, порушену дистанцію, спостереження або небезпечну тілесну правду, а не через нормальний обмін репліками
