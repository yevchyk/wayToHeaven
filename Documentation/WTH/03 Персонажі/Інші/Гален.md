# Гален

[[Гален]] це маєстр шахт, через якого пролог показує головне зіткнення між людським імпульсом і логікою системи.

## Що вже канонічно

- він змінив кріплення без наказу, щоб урятувати людей від обвалу
- він говорить коротко і по факту, бо для нього небезпека вже не абстракція, а виробнича реальність
- цей вчинок читається [[Батько|батьком]] як загроза дисципліні
- його покарання стає моральним переломом для [[Мірелла|Мірелли]]
- важливо, що він не грає героя і не просить милості: саме це робить сцену болючішою
