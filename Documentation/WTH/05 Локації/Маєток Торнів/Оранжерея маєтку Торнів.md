# Оранжерея маєтку Торнів

[[Оранжерея маєтку Торнів]] це простір красивої штучності. Вона підсилює мотив культивованого життя, яке існує лише завдяки контролю середовища.

## Чому це важливо

- це ідеальний простір для тихої правди після публічного ритуалу
- вона римується з темою форми без природності
- тут добре працюють сімейні й інтимні сцени розкриття

- [[Оранжерея]]
