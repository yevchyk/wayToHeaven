## Що треба розвивати далі

- Розкладати новий канон не в загальні чернетки, а в окремі сторінки.
- Для кожної нової сцени заводити окремий note за [[Шаблон сцени]].
- Для кожного нового персонажа заводити окремий note за [[Шаблон персонажа]].
- Для нових законів світу, систем і правил заводити notes за [[Шаблон механіки світу]].
